# TrustOS MVP

TrustOS is a trust infrastructure platform designed to simplify the process of finding and onboarding corporate trustees.

## Project Scaffold

This repository currently contains an initial scaffold for a minimum viable product (MVP):

- `backend/` — FastAPI backend placeholders.
- `frontend/` — React component placeholders for questionnaire and results UI.
- `database/` — SQLite schema placeholder.

## Directory Layout

```text
backend/
  main.py
  models.py
  matching.py

frontend/
  App.jsx
  Questionnaire.jsx
  Results.jsx

database/
  schema.sql
```

## Status

This is scaffolding only. Business logic, database integration, and full frontend behavior are intentionally not implemented yet.
